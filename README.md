# ERM Refresh Dashboard

## Host on GitHub Pages
1. Create a new GitHub repository.
2. Upload **`index.html`** (in this folder's root) to the repo root.
3. Go to **Settings → Pages → Source: Deploy from branch → `main` / `root`**.
4. Your site goes live at `https://<username>.github.io/<repo>/`.

`index.html` is fully self-contained — fonts, Tailwind, Chart.js, the logo, and all
data are embedded. It works offline and needs no other files to run.

## /source (optional, for editing)
The editable working file and its raw assets, in case you want to make changes:
- `ERM Refresh Dashboard.html` — the editable source
- `protiviti-logo-white.png` — logo
- `tailwind-v4-browser.js`, `fonts-inline.css` — embedded dependencies

You do **not** need the `/source` folder to host the site — only `index.html`.
